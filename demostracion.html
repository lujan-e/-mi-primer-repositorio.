Mi Proyecto Web

Introducción

Este proyecto es una demostración de cómo utilizar HTML para formatear y organizar contenido.

Código de Ejemplo

Para escribir código HTML, utiliza la etiqueta <code> para resaltar fragmentos de código.

"HTML es el lenguaje básico para construir páginas web."

Organización

Bloque de Contenido

El contenido dentro de un <div> se puede agrupar y estilizar en bloque.

Dentro de un párrafo, puedes usar etiquetas de formato para mejorar la presentación.
